## 版本 1：发朋友圈 / 发群 / 发小红书评论区式

做了一个自己真的会用的 Codex skill：

**wechat-content-router**

不是单独抓某个平台链接，
而是把**微信当成统一入口**：

- 微信里转来的小红书链接 → 自动落库
- 微信里转来的公众号链接 → 自动落库
- 可进 Obsidian
- 也可直接下载到本地文件夹

适合做：

- 知识库沉淀
- 资料归档
- 选题收集
- 微信 filehelper 内容中转

仓库：

`https://github.com/shenyj91/wechat-content-router`

如果你平时也靠微信转资料、收内容，这个应该能省不少手工活。

## 版本 2：更像 GitHub 首发介绍

发布了一个新的 Codex skill：

# wechat-content-router

A WeChat-first content router for Codex.

It turns WeChat-shared content links into structured local assets.

Current routes:

- XiaoHongShu → Markdown + images + optional OCR
- WeChat MP → Markdown + article body

Output modes:

- Obsidian vault
- Local folders

This is built for a very common real workflow:

1. forward useful content into WeChat
2. use WeChat/filehelper as the intake point
3. route links automatically
4. save them into your own system

Repo:

`https://github.com/shenyj91/wechat-content-router`
